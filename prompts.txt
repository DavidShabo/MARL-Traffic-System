## 🚨 **Critical Fix for Your Seed/Crash Issue**

Your training logs show repeated crashes:
```
AssertionError: scenario_index (seed) should be in [1000:2000)
```

**Cause**: `start_seed=1000` + `num_scenarios=1000` → seeds [1000:2000). Ray generates seeds that exceed this range.

**Fix**: Add `--start-seed 0` to all commands above **OR** update `build_env_config` in `env.py`:
```python
base = {
    ...
    "start_seed": 0,  # Changed from 1000
    "num_scenarios": 10000,  # Much larger range
}
```
